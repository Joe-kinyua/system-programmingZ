1. `cd`: Change directory. Used to navigate between directories.
2. `pwd`: Print working directory. Shows the current directory you are in.
3. `ls`: List directory contents. Lists files and directories in the current directory.
4. `mkdir`: Make directory. Creates a new directory.
5. `rmdir`: Remove directory. Deletes an empty directory.
6. `cp`: Copy files and directories.
7. `mv`: Move or rename files and directories.
8. `rm`: Remove files and directories.
9. `cat`: Concatenate files. Displays the contents of a file.
10. `less`: View file contents one page at a time.
11. `head`: Display the beginning of a file.
12. `tail`: Display the end of a file.
13. `grep`: Search for patterns in files.
14. `echo`: Print a message or value to the terminal.
15. `man`: Display the manual page for a command.
16. `history`: Show a list of previously executed commands.
17. `chmod`: Change file permissions.
18. `chown`: Change file ownership.
19. `chgrp`: Change group ownership.
20. `sudo`: Execute a command with superuser privileges.
